# Lottie-Web
本项目使用Vue3+TypeScript+WebAssembly在web端渲染矢量动画贴纸，可以实现Telegram中的贴纸效果。更多详细信息请查看文档。

![](https://upyun.liehuzuo.top/lottie.png)
- [文档](https://liehuzuo.top/archives/lottie-web文档)
- [在线演示](http://lottie.liehuzuo.top)

## 启动
### 本地开发环境
```
npm install
```

```
npm run dev
```
### 打包到生产环境
```
npm run build
```
